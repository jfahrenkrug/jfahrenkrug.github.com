I will make this post very short: RaphaelJS is amazing. It is a Javascript library that lets you easily create and manipulate vector graphics. I thought it would be great to have a beautiful Objective-J API to use it in Cappuccino projects. What I came up with is Raphuccino. You can find the code on github.
